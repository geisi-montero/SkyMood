# SkyMood

A weather web application built with HTML, CSS, and JavaScript that displays real-time weather information using the OpenWeatherMap API. The interface adapts dynamically to current weather conditions, providing an engaging visual experience.

---

## Features

- Real-time weather search by city.
- Dynamic themes based on weather conditions.
- Contextual mood messages.
- Responsive design for mobile and desktop.

---

## Technologies

- HTML5
- CSS3
- JavaScript (ES6+)
- OpenWeatherMap API
